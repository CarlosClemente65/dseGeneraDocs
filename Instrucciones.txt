- Para ejecutar el programa se debe pasar como parametros la clave de ejecucion de Diagram seguido del nombre del fichero con el guion de datos
	* dsegeneradocs.exe ds123456 guion.txt

- El guion esta dividido en secciones, y debajo de cada una se incluyen los datos correspondientes:
	* [parametros]	Seccion para indicar los parametros de entrada (de momento solo dos):
		- plantilla=	Fichero con la plantilla.docx a procesar
		- salida=		Nombre del fichero en el que se grabrara el resultado de la fusion
		- PDF=SI		Indica que se genere tambien el documento en PDF ademas del Word
		
	* [marcadores]	Seccion para incluir todos los marcadores que se sustituiran en la plantilla, poniendo el
					nombre del marcador entre el simbolo '#' seguido del signo de igual '=' y a continuacion 
					el texto que se sustituira.
					Ejemplo: #marcador#=Texto a sustituir
					Nota: se pueden poner espacios en el nombre del marcador, pero es aconsejable
					poner el guion bajo para que este unido en una palabra.
	* [tablas]		Seccion para indicar los textos a insertar en las tablas que pueda haber en el documento.
					Las tablas deben tener un encabezado y el texto de la primera celda (A1) sera el que se utilice 
					para localizarla entre todas. 
					Tiene que haber al menos una fila despues del encabezado que debe tener el formato que se quiere
					aplicar a cada una de las celdas (fuente, negrita, alineacion, etc) y que se utilizara para insertar
					todas las filas que sean necesarias segun los datos del guion. 
					Para añadir una tabla, en la primera linea debe indicarse un nombre entre el simbolo '#' seguido
					del signo igual '=', y a continuacion el texto que se buscara en la primera celda del encabezado.
					La siguientes lineas seran los datos que deben ir en cada celda separados con el simbolo '|'
					Ejemplo: #tabla1#=Entidad financiera
							 BBVA|HIPOTECARIO|100.000,00
							 LA CAIXA|PERSONAL|75.000,00
